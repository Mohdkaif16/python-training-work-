problem 78-subset
if set is [1,2,3] then subsets are {[],[1],[2],[3],[1,2],[2,3][1,3],[1,2,3]}
steps-
1. Break problem into smaller part
2. Get subsets of remaining elements
3. For each subset:
      - Keep it as it is
      - Add current element to it
4. Return all
Pseudocode with recursion-
FUNCTION subsets(nums):
    IF nums is empty:
        RETURN [[]]
    smallOutput = subsets(nums without first element)
    output = empty list
    FOR each list in smallOutput:
        ADD list to output
        ADD (first element + list) to output
    RETURN output
