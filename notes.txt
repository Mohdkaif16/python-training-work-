problem 78-subset
if set is [1,2,3] then subsets are {[],[1],[2],[3],[1,2],[2,3][1,3],[1,2,3]}
steps-
1. Break problem into smaller part
2. Get subsets of remaining elements
3. For each subset:
      - Keep it as it is
      - Add current element to it
4. Return all

Pseudocode with recursion-
FUNCTION subsets(nums):
    IF nums is empty:
        RETURN [[]]
    smallOutput = subsets(nums without first element)
    output = empty list
    FOR each list in smallOutput:
        ADD list to output
        ADD (first element + list) to output
    RETURN output
problem 90-subsets2
steps-
1. Sort the array
2. Generate all subsets (recursion)
3. Duplicates will appear
4. Remove duplicates
5. Return final answer

Pseudocode with recursion-
FUNCTION subsetsWithDup(nums):
    SORT nums
    FUNCTION solve(nums):
        IF nums is empty:
            RETURN [[]]
        small = solve(nums without first element)
        ans = empty list
        FOR each list in small:
            ADD list to ans
            ADD (first element + list) to ans
        RETURN ans
    result = solve(nums)
    final = empty list
    FOR each subset in result:
        IF subset not in final:
            ADD subset to final
    RETURN final
